# Отчет об экспериментах

**Дата генерации:** Запустите `poetry run python scripts/reports/generate_experiment_report.py` для генерации отчета

## Сводка

Для просмотра отчета об экспериментах запустите скрипт генерации:

```bash
poetry run python scripts/reports/generate_experiment_report.py
```

Отчет будет сохранен в `reports/experiments/latest.md` и будет включать:

- Сравнительную таблицу всех экспериментов
- Графики сравнения метрик
- Детальную информацию о каждом эксперименте
