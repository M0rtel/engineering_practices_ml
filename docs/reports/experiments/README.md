# Отчеты об экспериментах

Эта директория содержит автоматически сгенерированные отчеты об экспериментах.

## Генерация отчетов

Для генерации отчета об экспериментах:

```bash
poetry run python scripts/reports/generate_experiment_report.py
```

Это создаст:
- `reports/experiments/latest.md` - последний отчет в формате Markdown
- `reports/experiments/metrics_comparison.png` - график сравнения метрик
- `reports/experiments/model_comparison.png` - график сравнения моделей

## Автоматическая генерация

Отчеты автоматически генерируются при:
- Запуске всех экспериментов: `poetry run python scripts/experiments/run_all_experiments.py`
- Запуске пайплайна с мониторингом: `poetry run python scripts/pipeline/run_pipeline.py --monitor`

## Структура отчета

Отчет включает:

1. **Сводка** - статистика по моделям и топ-5 экспериментов
2. **Сравнительная таблица** - полная таблица всех экспериментов
3. **Визуализации** - графики сравнения метрик и моделей
4. **Детали экспериментов** - подробная информация о каждом эксперименте

## Просмотр отчетов

Отчеты доступны в веб-интерфейсе документации: [Latest Report](latest.md)
